# NAME

pgp-fingerprint - Prints the fingerprint of the PGP public key inside the supplied file

## Example

    ./pgp-fingerprint /path/to/public.key
    3f38251ac169da0f087b2479a3384b233c92aeee

# Exit Status:

* 0 if OK
* 1 if there was an error
